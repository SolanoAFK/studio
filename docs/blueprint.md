# **App Name**: AquaLogin

## Core Features:

- Email Login: Allow users to log in using their email address and password.
- Password Reset: Enable users to reset their password if they forget it.
- Account Creation: Allow new users to create an account with their email and password.
- Session Management: Manage user sessions to keep them logged in securely.

## Style Guidelines:

- Primary color: Soft, professional blue (#64B5F6) evoking water and clarity.
- Background color: Very light blue (#E3F2FD), almost white, creating a clean, fresh backdrop.
- Accent color: Teal (#4DB6AC), providing a subtle contrast while remaining within the water theme.
- Font: 'Inter' (sans-serif) for a clean and professional look. Suitable for both headings and body text.
- Use minimalist icons with a water droplet or wave motif to reinforce the theme.
- Subtle animations, such as a gentle ripple effect on buttons or a smooth transition between login and registration forms, to enhance the user experience.
